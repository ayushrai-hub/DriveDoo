üé® Color Contrast Improvements

Current Issues:
  • Blue text on white background: 3.2:1 ratio (needs 4.5:1)
  • Yellow accents may not meet contrast requirements
  • Hover states need sufficient contrast

Recommended Color Palette

/* Enhanced Color Palette for Better Contrast */ :root { --primary-dark: #0d2a4d; /* Darker blue for better contrast */ --primary-accent: #f9bf3f; /* Keep yellow (meets contrast) */ --text-dark: #222222; /* Darker text */ --text-light: #ffffff; /* White text */ --bg-light: #ffffff; /* White background */ --border-contrast: #cccccc; /* High contrast borders */ } /* High Contrast Mode Support */ @media (prefers-contrast: high) { :root { --primary-dark: #000000; --text-dark: #000000; --bg-light: #ffffff; } }
